### 技術選型

專案以 Spring AI 開發框架實戰為核心，涉及到多種主流 AI 客戶端和工具庫的運用。

- Java 21 + Spring Boot 3 框架
- ⭐️ Spring AI + LangChain4j
- ⭐️ RAG 知識庫
- ⭐️ PGvector 向量數據庫
- ⭐ Tool Calling 工具調用 
- ⭐️ MCP 模型上下文協議
- ⭐️ ReAct Agent 智能體構建
- ⭐️ Serverless 計算服務
- ⭐️ AI 大模型開發平台百煉
- ⭐️ Cursor AI 程式碼生成
- ⭐️ SSE 異步推送
- 第三方接口：如 SearchAPI / Pexels API
- Ollama 大模型部署
- 工具庫如：Kryo 高性能序列化 + Jsoup 網頁抓取 + iText PDF 生成 + Knife4j 接口文件
